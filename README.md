# Emoji web app (Python)

Минимальное веб‑приложение на Flask, которое показывает большой эмодзи по центру экрана.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Откройте: http://localhost:8000

Чтобы было доступно в сети, приложение слушает `0.0.0.0:8000`.
